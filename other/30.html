配置Apache使用局域网共享目录（Windows）

  • 最新
  • 时间线

打开CMD，创建硬链接 

mklink /D D:\Web\myshare //10.0.0.2/myshare

配置Apache


ServerName myshare.com
DocumentRoot "D:/Web/myshare"

Options FollowSymLinks ExecCGI
AllowOverride All
Order deny,allow
allow from all

如果网站访问出现问题，或者提示无法遵循符号链接，可能是符号链接权限的问题，请在CMD输入以下命令

fsutil behavior set SymlinkEvaluation L2L:1 L2R:1 R2L:1 R2R:1
